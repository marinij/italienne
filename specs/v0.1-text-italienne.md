# v0.1 — Text-Only Italienne

> **Status**: current
> **Goal**: A working italienne rehearsal from pasted text — the simplest useful version.

## User Story

As an actor, I want to paste a scene's text, pick my character, and rehearse my lines with an AI giving me the cues — so I can practice alone without a partner.

## Scope

This spec needs to be detailed by an agent. The high-level requirements are:

### Input
- Paste raw text of a theater scene into a text area
- The parser extracts characters and their dialogue lines
- User selects which character they're playing

### Rehearsal
- Lines are presented in order
- Other characters' lines are displayed as cues
- User types their own character's lines from memory
- User can reveal/skip a line if stuck

### Review
- At the end, show a fidelity report:
  - Per-line comparison (expected vs actual)
  - Accuracy score (word-level diff)
  - Highlighted differences (missing words, additions, substitutions)

## Technical Notes

- All client-side for v0.1 — no API calls needed
- Script parsing is the hardest part: theater scripts have no standard format
- Start with a simple heuristic parser (character name in caps followed by dialogue)
- Store session in React state, no persistence needed yet

## Open Questions

- What text formats should we support? (French theater conventions, English, mixed?)
- Should we support multiple scenes in one paste?
- How strict should the fidelity scoring be? (exact match vs semantic similarity)
